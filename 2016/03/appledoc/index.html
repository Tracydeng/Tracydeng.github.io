iOS-appledoc文档添加

一.安装过程

  • 下载安装

    git clone git://github.com/tomaz/appledoc.git
cd ./appledoc
sudo sh install-appledoc.sh

    稍等一会，安装OK

  • 验证安装

    appledoc --version

二.使用

  • 进入工程代码所在文件夹：打开终端

    cd + "文件夹目录"  //直接拖到终端就行

  • 输出文档

  1. project-name:项目名称
  2. project-company:公司名称
 appledoc --project-name MaizuoMovie --project-company HYX ./

之后你会在项目目录下多处一个文件，稍后会说这个文件的作用：

再到Xcode－>Window->Documentation and API Refrence，你就已经可以查看到项目输出的文档了，和苹果官方文档一样的格式

如果想了解更多， 可以查看帮助：

 appledoc --help

  • 集成进我们的工程

  • 在你的工程中创建新的Target，选择Other中的Aggregate,如图：

  • 在新创建的Target中的Build Phases中添加Run Script，如图

  • 打开Run Script, Shell下面的文档区域添加这样的模版

    #appledoc Xcode script  
# Start constants  
company="ACME";  
 companyID="com.ACME";
 companyURL="http://ACME.com";
target="iphoneos";
#target="macosx";
outputPath="~/help";
# End constants
/usr/local/bin/appledoc \
--project-name "${PROJECT_NAME}" \
--project-company "${company}" \
--company-id "${companyID}" \
--docset-atom-filename "${company}.atom" \
--docset-feed-url "${companyURL}/${company}/%DOCSETATOMFILENAME" \
--docset-package-url "${companyURL}/${company}/%DOCSETPACKAGEFILENAME" \
--docset-fallback-url "${companyURL}/${company}" \
--output "${outputPath}" \
--publish-docset \
--docset-platform-family "${target}" \
--logformat xcode \
--keep-intermediate-files \
--no-repeat-first-par \
--no-warn-invalid-crossref \
--exit-threshold 2 \
"${PROJECT_DIR}"
  • Xcode左上方选择这个Target,然后Build编译
  • 文档就会编译好并自动安装进Xcode了（重启Xcode生效）。
  • 生成的文／Mitchell（简书作者）docset-installed.txt文件啦，打开它：看到以一个 Path 了吧：直接复制，然后在 Finder 中前往文件夹：
  • 前往文件夹，就可以看到,右键－>显示包内容，就可以看到所有注释导出来的文档了

	/*! 同样是单行注释 */
	/** 这也是单行注释，
	*  第二行会接上第一行。
	*/
